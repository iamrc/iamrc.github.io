hexo+github搭建我的博客

发表于 2017-01-07

安装

安装node

用来生成静态页面 到node.js官网下载相应平台的最新版本，一路安装。

安装git

用来把本地的hexo内容提交到github上去.

申请github账户

是用来做博客的远程创库、域名、服务器之类的，（怎么与本地hexo建立连接等下讲) github直接申请就行了，ssh keys，不配置的话以后每次对自己的博客有改动提交的时候就要手动输入账号密码，配置了就不需要了，怎么配置，到网上找教程。

安装hexo

npm安装
$ npm install -g hexo
cnpm安装（建议用cnpm国内包套镜像会快很多，怎么配置cnpm，到网上找教程）
$ cnpm install -g hexo

初始化

hexo init 文件夹名称
完成后，长这个样子
Blog 　｜ 　｜－－ _config.yml 　｜－－ node_modules 　｜－－ public 　｜－－ source 　｜－－ db.json 　｜－－ package.json 　｜－－ scaffolds 　｜－－ themes

启动

本地启动

hexo sercer (或hexo s)
然后，浏览器输入http://localhost:4000,就可以看到自己的博客了

配置Github

建一个github仓库

注意github账户名为https://github.com/**username**仓库名必须为**username**.github.io，固定写法

建立关联并用git部署

修改config.yml文件中的deploy,这里如果你设置了ssh，repository这里直接在github里复制SSH的就行了,每次对自己的博客有改动提交的时候就不用手动输入账号密码
deploy: type: git repository: https://github.com/username/username.github.io.git
执行如下命令，下载一个部署插件，才能使用git部署
npm install hexo-deployer-git --save
好了，用git部署
hexo deploy（或者hexo d）
总结一下，每次部署
hexo clean hexo generate hexo deploy

#新建文章并部署
hexo new "postName" #新建文章 hexo generate(或hexo g) #生成静态页面至public目录 hexo server(或hexo s) #开启预览访问端口（默认端口4000，'ctrl + c'关闭server） hexo deploy(或hexo d) #将.deploy目录部署到GitHub
